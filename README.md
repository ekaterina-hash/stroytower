# 🪵 Строй Тауэр — Интернет-магазин пиломатериалов

## Структура проекта

```
stroytower/
├── index.html              # Главный файл (все страницы — SPA)
├── .htaccess               # Apache-конфигурация
├── assets/
│   ├── css/
│   │   └── style.css       # Стили
│   └── js/
│       └── main.js         # JavaScript (каталог, корзина, роутинг)
├── php/
│   ├── contact.php         # Обработчик формы обратной связи
│   └── order.php           # Обработчик оформления заказа
└── logs/                   # Логи (создаётся автоматически, .gitignore)
```

## Установка

### 1. Загрузить файлы на хостинг
Скопируйте все файлы в корневую директорию сайта (public_html / www / htdocs).

### 2. Настроить email в PHP
Отредактируйте `php/contact.php` и `php/order.php`:
```php
define('ADMIN_EMAIL', 'ваш@email.ru');
```

### 3. Проверить работу PHP
- Загрузите `php/contact.php` на сервер
- Убедитесь, что PHP ≥ 7.4 и функция `mail()` включена
- Для надёжной отправки рекомендуется PHPMailer + SMTP

### 4. Изменить контактные данные
Найдите в `index.html` и замените:
- `8 800 123-45-67` → ваш номер телефона
- `info@stroytower.ru` → ваш email
- `г. Москва, ул. Примерная, д. 1` → ваш адрес
- `ООО «Строй Тауэр»` → ваше юридическое название
- ИНН, ОГРН, КПП → ваши реквизиты

### 5. Добавить товары
В файле `assets/js/main.js` в массиве `PRODUCTS` добавьте/измените товары:
```javascript
{
    id: 9,                          // уникальный ID
    slug: 'brus-200x200',           // URL-идентификатор
    name: 'Брус 200×200',           // название
    category: 'Брус',               // категория
    categorySlug: 'brus',           // slug категории
    dims: '200×200×6000 мм',        // размеры
    price: 1800,                    // цена за пог.м
    priceUnit: '₽/пог.м',
    badge: null,                    // null / 'new' / 'sale' / 'popular'
    desc: 'Описание товара...',
    specs: {
      'Порода дерева': 'Сосна',
      // ...
    }
}
```

## Правовые документы

Сайт содержит все необходимые документы в соответствии с законодательством РФ:

- **Политика конфиденциальности** — ФЗ от 27.07.2006 № 152-ФЗ «О персональных данных»
- **Публичная оферта** — ст. 435 ГК РФ, Постановление Правительства РФ от 27.09.2007 № 612
- **Пользовательское соглашение** — регулирует использование сайта

⚠️ **Важно**: Перед запуском замените все шаблонные данные (ИНН, ОГРН, адрес, реквизиты) на реальные и проконсультируйтесь с юристом.

## Технические требования

- PHP 7.4+ (для форм)
- Apache с mod_rewrite
- Поддержка функции `mail()` или SMTP-сервер

## Без PHP (только HTML/JS)

Сайт работает как SPA (Single Page Application) без сервера. Формы в этом случае показывают успешное сообщение без реальной отправки. Для отправки форм без PHP можно использовать сервисы Formspree, EmailJS и др.
